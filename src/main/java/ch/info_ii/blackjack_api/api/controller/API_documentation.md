# Endpoints Documentation
| Méthode | Endpoint                  | Body | Parameters  | Response                                                                                                                                     | Description                                                                                                                                  |
|---------|---------------------------|------|-------------|----------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------|
| GET     | /                         | {}   |             | blackjackinterface.html                                                                                                                      | Interface du jeu                                                                                                                             |
| POST    | /api/start                | {}   |             | {}                                                                                                                                           | Débute le jeu. Reste à charger durant tout le jeu. Pour l'appel d'un autre endpoint, on peut le cancel ou appeler dans une nouvelle fenêtre. |
| POST    | /api/new_player           | {}   | name=String | {"nom" : String<br/>,"jetons" : int,<br/>"score" : int<br/>"hand" : [{"couleur" : String,<br/>"valeur" : int},...],<br/>"cardsNumber" : int} | Création d'un nouveau joueur. Le nom du joeur est à donner en paramètre.                                                                     |
| GET     | /api/player               | {}   |             | {"nom" : String<br/>,"jetons" : int,<br/>"score" : int<br/>"hand" : [{"couleur" : String,<br/>"valeur" : int},...], "cardsNumber" : int}     | Obtenir les infos du joueur                                                                                                                  |
| POST    | /api/player/bet_tokens    | {}   | number=int  | {"number" : int}                                                                                                                             | Miser des jetons. Le nombre des jetons à miser est à donner en paramètre.                                                                    |
| GET     | /api/current_party        | {}   |             | {"etat" : String,<br/>"taux" : int,<br/>"nombreDeJetonsMises" : int,<br/>"gagnant" : String,<br/>"finished" : Boolean}                       | Retourne des informations sur la partie actuelle                                                                                             |
| PUT     | /api/player/play/new_card | {}   |             | {"nom" : String<br/>,"jetons" : int,<br/>"score" : int<br/>"hand" : [{"couleur" : String,<br/>"valeur" : int},...],<br/>"cardsNumber" : int} | Le joureur tire une nouvelle carte                                                                                                           |
| PUT     | /api/player/play/stay     | {}   |             | {"nom" : String<br/>,"jetons" : int,<br/>"score" : int<br/>"hand" : [{"couleur" : String,<br/>"valeur" : int},...],<br/>"cardsNumber" : int} | Le joueur ne tire pas de carte                                                                                                               |
| PUT     | /api/player/play/abandon  | {}   |             | {"message" : String}                                                                                                                         | Le joueur abantone le jeu                                                                                                                    |
| GET     | /api/croupier/score       | {}   |             | {"score" : int,<br/>"cardsNumber" : int,<br/>"firstCard" : {"couleur" : String,<br/>"valeur" : int}}                                         | Obtenir le score du croupier                                                                                                                 |
| GET     | /api/croupier             | {}   |             | {"nom" : String<br/>"score" : int<br/>"hand" : [{"couleur" : String,<br/>"valeur" : int},...],<br/>"limite" : int,<br/>"cardsNumber" : int}  | Obtenir la main du croupier                                                                                                                  |
| GET     | /api/new_party            | {}   |             | {}                                                                                                                                           | Commencer une nouvelle partie, après avoir déjà fini une                                                                                     |
| PUT     | /api/empty_hands          | {}   |             | {}                                                                                                                                           | Vide les mains des joeurs, utilisé en fin de parties                                                                                         |
